# Family Care Planner

A lightweight family care scheduling board for Mo and Lisa.

## Features

- Drag people into care slots
- Switch assignees from a slot menu
- Add notes to each slot
- Undo and redo changes
- Export assignments as an `.ics` calendar file

## Local Use

Open `index.html` in a browser.

## GitHub Pages

This project is designed to run as a static site from GitHub Pages.
